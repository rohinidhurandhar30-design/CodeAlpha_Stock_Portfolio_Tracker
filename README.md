# Stock Portfolio Tracker

## Description

A simple Python application that calculates the total investment value of a stock portfolio using predefined stock prices.

## Features

- User inputs stock symbol and quantity
- Uses predefined stock prices
- Calculates total investment value
- Saves portfolio details into a text file
- Simple console-based interface

## Technologies Used

- Python 3
- Dictionary
- Loops
- Input/Output
- File Handling

## Available Stocks

| Stock | Price |
|--------|------:|
| AAPL | $180 |
| TSLA | $250 |
| GOOGL | $140 |
| MSFT | $330 |
| AMZN | $150 |

## How to Run

```bash
python stock_tracker.py
```

## Output

The portfolio summary is displayed on the console and saved in `portfolio.txt`.
